## рҹҡҖ Quick Start

**Use it online вҖ” no install needed:**

```
https://lebedevnet.github.io/ReadmeForge/
```

1. Fill in your name, role & bio
2. Select your tech stack (click to toggle)
3. Add social links
4. Pick a theme & enable widgets
5. Click **вңЁ Generate README**
6. Copy вҶ’ paste into your GitHub profile repo

**Or run locally:**

```bash
git clone https://github.com/lebedevnet/ReadmeForge.git
cd ReadmeForge
open index.html
```

> No npm. No build step. No dependencies. Just open the file.
